6.git & git-hub 연결 및 push 하는 방법
1.초기화 : git init
2.연결작업 : git remote add origin "레포지토리 주소"
- git remote -v
3.스테이징 저장공간 활용 : git add .
- git status
4.커밋메세지 작성 : git commit -m "메세지이름"
5.푸쉬 완료 : git push origin master
