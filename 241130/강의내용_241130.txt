아이디선택자 > 태그선택자 > 전체선택자 가장 하위 순위
cheat key : 

- 선택자!!!
1) 전체선택자 : * (wild card selector) / 문서상에 있는 모든 태그들에게 공통적인 스타일을 적용하고 싶을 때

2) 태그선택자 : 특정 태그를 찾아서 해당 태그에만 어떤 스타일을 적용하고 싶을 때

3) 아이디선택자 : 어떤 문서를 제작할 때, 고유한 값을 가지고 있는 요소를 만들어서 해당 요소한테만 특별한 스타일을 적용하고 싶을 때, 사용할 수 있는 선택자 => 무조건 한 문서상에서 한 번 밖에 사용할 수 없음!!

4) 클래스선택자 : 문서상에서 동일한 스타일을 여러개의 요소들에게 동시에 적용하고자 할 때

5) 후손선택자 : 할머니, 할아버지 <-> 손주

6) 자손선택자 : 직계부모 <-> 직계자식

7) 동위선택자 : 한 부모 밑에 같은 위치에 존재하는 형제들사이에 어떤 스타일을 적용하고자 할 때 (+ / ~)

8) 구조선택자 : 
- 일반구조선택자 : 한 부모 밑에 같은 위치에 존재하는 형제들 중 짝수, 홀수, 가장 첫번째, 가장 마지막번째, 특정 번째의 형제들을 찾아오고자 할 때 (*한 부모 밑에 있는 형제들이 모든 동일한 태그를 유지)
nth-child / n || even || odd
first-child
last-child

- 형태구조선택자 : 한 부모 밑에 같은 위치에 존재하는 형제들 중 짝수, 홀수, 가장 첫번째, 가장 마지막번째, 특정 번째의 형제들을 찾아오고자 할 때 (*한 부모 밑에 있는 형제들의 태그들이 상이한 경우)
nth-of-type / n || even || odd
first-of-type
last-of-type

9) 반응선택자 : 사용자의 이벤트 행위에 의한 스타일을 적용해주고자 할 때
:hover
:active

10) 상태선택자 : form 요소 태그들의 상태에 따라서 스타일을 상이하게 적용해주고자 할 때
:enabled
:disabled
:focus

11) 속성선택자 : form 요소 태그들의 속성값을 찾아와서 스타일을 적용하고자 할 때 

12) 링크선택자 : a태그 가운데, 링크를 가지고 있는지에 대한 여부에 따른 스타일을 적용하고자 할 때

13) 부정 (not)선택자 : 특정 요소가 아닌 값들을 선택해서 스타일을 적용하고자 할 때

14) 문자선택자 : 특정 요소 앞.뒤에 스타일을 적용 (*전.후 문자선택자)
