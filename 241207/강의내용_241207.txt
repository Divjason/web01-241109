CSS 속성 position

- static : 정적인 <-> 동적인 // html 태그를 작성하면 원래 작성한 html 순서에 맞춰서 웹 브라우저 화면에 출력되게끔 설정!!!
> 개발자가 굳이 position 설정을 하지 않아도 기본값으로 설정이 되어 있음!!!

- relative : 상대적인 <-> 절대적인 // 원래 본인 요소가 위치해있어야 하는 해당 포지션을 기준으로 상대적으로 얼만큼 떨어져 있게 할 것 인가를 결정!!!

- absolute : 절대적인 // 다른 형제 요소들의 위치와 상관없이 브라우저를 기준으로 절대적인 좌표값을 갖는다. 단, 부모요소 중에 position: relative 값을 갖고 있는 요소가 있다면, 해당 부모를 기준으로 절대 좌표값을 가져갈 수 있다. 그리고, 조부모를 기준으로 position:relative값을 적용한 후 부모요소에서 position: absolute값을 적용한 이후 자식요소에서 또 position: absolute값을 적용할 수 있다!!!

- fixed : 고정된 // 마우스 스크롤 휠을 위.아래로 움직이더라도 해당 요소는 절대로 위치가 변하지 않는다

- sticky : sticker에서 유래한 형용사 // 평상시에는 일반적인 아이템들처럼 이동하다가 특정 포지션 위치에 도착하면 마치 스티커처럼 부착되어지는 효과를 적용
