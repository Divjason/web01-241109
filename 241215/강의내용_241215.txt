*전환(transition) 효과
A라는 아이템을 B라는 아이템으로 변형을 시키고자 하는데, 그 때 변형 단계에서 부드럽고, 자연스럽게 변형하고자 할 때 사용할 수있는 기능

transition-property
transition-duration
transition-delay
transition-timing-function


*애니메이션(animation) 효과
- 어떠한 애니메이션을 만들고자 할 때, 속성을 정의할 수 있도록 해주는 기능 // 반드시 애니메이션 정의 => 애니메이션 이름을 작명 후 속성 값을 부여!!

animation-name
animation-duration
animation-timing-function
animation-delay
animation-iteration-count
animation-direction

> frame : 어떤 모션 혹은 영상을 만들기 위해서는 정적인 형태의 개별적인 사진이 모여야 함!!!
> 총 30장의 사진이 모여야 인간이 어색함을 느끼지 않을 정도의 영상 // 30프레임 // 29.97 => 1초 동영상 

*변환(transform) 효과
A라는 아이템을 B라는 아이템으로 변형을 할 때, 어떻게 변형할 것인가를 고민하고, 기능을 부여!!!
> translate : 원래 아이템이 위치해있는 포지션의 값을 변환
> rotate : 원래 아이템을 회전
- 양수값 : 시계방향으로 회전
- 음수값 : 반시계방향으로 회전

> scale : 원래 아이템의 크기
- 원래 해당 아이템이 가지고 있는 사이즈 100% = 1

> skew : 원래 아이템의 기울임


=> 주의사항 : 동일한 속성값이라도 무엇을 먼저 작성하는가에 따라서 최종 결과값이 완전히 다르게 나올 수 있음!
